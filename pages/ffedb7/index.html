多行展开收起自动隐藏

# placeholder 样式修改

placeholder 作为 input 占位符，经常在表单中使用，但你知道吗？其实 placeholder 样式也是能够通过 css 来修改的

支持的属性

  • background properties
  • color
  • font properties
  • letter-spacing
  • line-height
  • opacity
  • text-decoration
  • text-indent
  • text-transform
  • vertical-align
  • word-spacing
<html>
	<input class='input1' placeholder="CSS Placeholder1">
	<input placeholder="CSS Placeholder2">
</html>

<style>
.input1::placeholder {
  color: deeppink;
}
</style>
