# еӯ—з¬ҰдёІзҡ„жӯЈеҲҷж–№жі•

ES6 еҜ№жӯЈеҲҷиЎЁиҫҫејҸж·»еҠ дәҶuдҝ®йҘ°з¬ҰпјҢеҗ«д№үдёәвҖңUnicode жЁЎејҸвҖқпјҢз”ЁжқҘжӯЈзЎ®еӨ„зҗҶеӨ§дәҺ\uFFFFзҡ„ Unicode еӯ—з¬ҰгҖӮд№ҹе°ұжҳҜиҜҙпјҢдјҡжӯЈзЎ®еӨ„зҗҶеӣӣдёӘеӯ—иҠӮзҡ„ UTF-16 зј–з ҒгҖӮ
