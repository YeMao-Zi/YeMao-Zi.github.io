# 引入css样式

## 方法一

在入口(app)文件的样式中引入

~~~html
<style>
	@import url('./assets/css/baseCss.css');
</style>
~~~

## 方法二

在html文件中以标签引入

~~~html
<link rel="stylesheet" href="<%= BASE_URL %>css/baseCss.css">
~~~

## 方法三

在main.js中通过引入文档的方式引入

> 在JS中引入图片或样式不需要from

~~~javascript
import './assets/css/baseCss.css'
~~~
